# Gemini-2.5-Flash-Chatbot-App
A modern, interactive web application built with Streamlit and the Google GenAI SDK that leverages the power of the Gemini 2.5 Flash model for real-time, engaging conversational AI. This project demonstrates best practices for building dynamic, stateful chatbots using Python and Streamlit's latest UI features.

- Gemini 2.5 Flash: Built on Google's fastest and most capable lightweight model, optimized for speed and high-volume tasks.
- Modern Streamlit UI: Features a dark-mode theme, a clean column layout for controls (Title/Clear Chat), and a dedicated "Clear Chat" button for enhanced user control.
- Session Management: Correctly uses st.session_state to store and manage the continuous chat flow across Streamlit reruns.
